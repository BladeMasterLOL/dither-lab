# Dither Lab

Aplicación web local para convertir imágenes en arte con dithering. Incluye cuatro algoritmos, cinco paletas, ajustes en vivo, comparación antes/después y exportación PNG.

## Sitio público

https://blademasterlol.github.io/dither-lab/

## Ejecutar

Abre `index.html` directamente en el navegador o sirve la carpeta con cualquier servidor estático.

Todo el procesamiento ocurre en el navegador. Las imágenes no se suben a ningún servidor.
